================
django-email-kit
================

Required:
=========

- Django==1.3.1 (may work in other versions)
- django-uni-form==0.9
- django-simple-captcha==0.3.0

Running the tests:
==================

Additional requirements:
------------------------

- dingus==0.3.3

Use the example_site or install the app into your own and run
``manage.py test``.
